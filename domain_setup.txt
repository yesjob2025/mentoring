1. CNAME
2. robots.txt
3. sitemap.xml
4. nginx.conf (직접 서버 배포 시)

[수정할 값]
- your-domain.com 을 실제 도메인으로 변경
- www 사용 시 your-domain.com / www.your-domain.com 둘 다 설정

[배포 가능 방식]
1. Netlify: 폴더 전체 업로드
2. Vercel: 폴더 전체 업로드
3. Nginx 서버: index.html + nginx.conf 사용
4. Apache 서버: index.html + .htaccess 사용
5. Docker 서버: Dockerfile / docker-compose.yml 사용
